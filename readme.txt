=== Description ===

The TopBar plugin adds a notification bar on your website to highlight your newest products, promotions, or upcoming events. This enables you to showcase a pleasant message to your visitors along with a custom link.

Display a customized top bar and make it visible for everyone or only logged-in users. Customize the appearance of the top notification bar with unique settings such as bar position, background color, text color, customized button, personalized link, and top bar width and height. The top bar will help you to inform your customers about the latest promotions to increase user engagement.

== Plugin Features ==
* Show the notification bar at the top of the page
* Show the top bar to everyone or logged-in users
* Option to make top bar sticky (fixed)
* Change the background and text color of the notification bar
* Add a personalized message to the top bar
* Include a CTA button with a custom linkreadme validator
* Customize text color, background color, and add custom CSS

== Installation ==
1. In your WordPress admin panel, go to Plugins > New Plugin
2. Find our TopBar plugin by Fmeaddons and click Install now
3. Alternatively, download the plugin and upload the contents of topbar.zip to your plugins directory, which usually is /wp-content/plugins/
4. Activate the plugin
